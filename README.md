Versão estática pronta para GitHub e Vercel.

## Arquivos

O pacote contém somente:

- `index.html` — aplicação completa, com CSS, JavaScript e imagens incorporados;
- `README.md` — instruções de publicação.

## Publicar no GitHub

1. Crie um repositório novo.
2. Extraia o ZIP.
3. Envie `index.html` e `README.md` para a raiz do repositório.
4. Faça o commit.

Não é necessário enviar pastas `src`, `assets`, `public` ou `dist`.

## Publicar no Vercel

1. Acesse o Vercel.
2. Clique em **Add New → Project**.
3. Importe o repositório do GitHub.
4. Em **Framework Preset**, escolha **Other**.
5. Não use comando de build.
6. Não altere o diretório de saída.
7. Clique em **Deploy**.

- As quatro imagens de capa estão incorporadas no `index.html`.
- As miniaturas e os players dependem da internet e do YouTube.
